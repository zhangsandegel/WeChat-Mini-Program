这是一个微信小程序

使用 JS/XML/CSS 进行界面设计，功能设计是将传感器连接蓝牙设备，
通过蓝牙获取传感器的数据，并进行图表展示。再将传感器数据传输到服务器，并在服务器端
调用机器学习算法，对数据进行处理，再返回小程序。实现了界面化设计和功能处理
